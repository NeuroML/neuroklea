## Errors and Lessons
- Docling's `HybridChunker` heading format (clean heading text, no `#` markers) is compatible with the existing URL map keys -- confirmed via test on `nml-elife` and `nml-docs` sources.
- `DocMeta.headings` is `list[str] | None` -- need `get("headings") or []` to handle None properly in metadata lookup.
- Python module-level imports execute eagerly: Typer needs only the function signature for `--help`, but importing the function's module forces all its top-level imports too. Lazy-import inside the function body solves this.
- No unicode anywhere -- `AGENTS.md` and comments must use ASCII only.
