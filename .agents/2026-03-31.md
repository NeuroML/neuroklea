# Session Log: 2026-03-31

## Summary

Extracted shared infrastructure from `rag_pkg` and `code_ai_pkg` into `utils_pkg` to reduce duplication. Both packages now use common base classes for orchestrators, nodes, and vector stores.

### Base Classes Extracted

- **`BaseLangGraph`** (`graph.py`) — Orchestrator base: config loading, MCP client, logging, checkpointing, MCP tool listing, vector store setup, domain schema creation, graph execution
- **`VectorStores`** (`stores.py`) — ChromaDB vector store management with domain-based retrieval
- **`BaseLangGraphNode`** (`nodes/base_nodes.py`) — Root ABC for all nodes; generic over return type (`Dict` for state-updating nodes, `str` for routers)
- **`BaseLLMNode`** — Template method pattern for LLM-calling nodes with `_pre_exec()` guard hook
- **`BaseMemoryLLMNode`** — Adds file-based prompt loading and memory support
- **`BaseRouterNode`** — Base for routing nodes returning `str`
- **`SummariseMemoryNode`** (`nodes/summarise_memory.py`) — Concrete node extracted from RAG
